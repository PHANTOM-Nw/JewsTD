### 伤害计算公式

```typescript
// 物理伤害计算
actualDamage = damage * (1 - armor / (armor + 10))

// 魔法伤害计算  
actualDamage = damage * (1 - magicResist)

// 纯粹伤害计算
actualDamage = damage  // 无视任何减免
```

### 自定义建议

**降低敌人难度**:
- 减少生命值(-10%~-20%)
- 移除护甲和魔抗
- 减慢移动速度(基础敌人40-50)

## 📈 等级系统

### 等级名称和倍率

| 等级 | 名称 | 图标 | 伤害倍率 | 范围倍率 | 攻速倍率 |
|------|------|------|---------|---------|---------|
| **Chipped** | 碎裂 | C | 1.0x | 1.0x | 1.0x |
| **Flawed** | 有瑕 | F | 1.2x | 1.05x | 1.0x |
| **Normal** | 普通 | N | 1.6x | 1.1x | 1.0x |
| **Flawless** | 无瑕 | L | 2.2x | 1.15x | 1.0x~1.2x |

### 升级规则

- **基础合成**: 2个相同类型+等级的塔 → 升级为下一等级
- **最高等级**: 无瑕(Flawless),无法继续升级
- **特殊塔**: 合成时等级 = min(材料1等级, 材料2等级)

### 波次设计原则

1. **教学波(1-2)**: 少量基础敌人,让玩家熟悉操作
2. **引入波(3-5)**: 逐步引入新敌人类型
3. **挑战波(6-9)**: 数量和强度稳步提升
4. **极限波(10-12)**: 高密度敌人,考验玩家实力

- 困难模式: 所有敌人数量×1.3,生命值×1.2,护甲+2

## 🔧 如何应用修改

**文件**: `src/config/enemies.ts`

  reward: number      // 修改奖励

    health: 50,      // ← 修改这里

    reward: 10
