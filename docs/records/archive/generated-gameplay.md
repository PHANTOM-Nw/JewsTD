## 📖 游戏简介

宝石TD(Gem TD)是一款基于Dota2游廊经典地图改编的塔防游戏。与传统塔防不同,宝石TD采用**无固定路径**的设计,玩家需要在地图上建造迷宫来引导敌人走向矿坑。

### 核心特色
- 🗺️ **无固定路径**: 自由建造塔和障碍物,创造迷宫引导敌人
- 💰 **双资源系统**: 木材用于建造,金币用于升级
- 🔧 **跨波次合成**: 保留的塔可以累积,支持多波次后合成高级塔
- ⚔️ **塔特性差异化**: 每种宝石塔有独特的攻击特性
- 🎯 **保护矿坑**: 敌人必须经过中心矿坑,保护它不被摧毁

## 🚀 快速开始

### 运行游戏
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:5173/
```

### 基本流程
1. 点击地图空地放置塔(消耗木材)
2. 放置5个塔后选择1个保留,其余变障碍
3. 点击"开始第X波"按钮
4. 敌人沿路径移动,塔自动攻击
5. 波次完成后继续放置,循环进行

## 🎮 基础操作

### 放置塔的流程

#### 第一步: 点击空地放置塔
- 点击地图上的**灰色空格子**
- 每次随机生成1个宝石塔(紫水晶/钻石/黄玉/蛋白石)
- 消耗1点木材
- 每波最多放置5次

#### 第二步: 选择保留哪个塔
- 放置完5个塔后,会弹出决策对话框
- 点击其中一个塔选择保留
- **保留的塔**: 留在原位继续显示和攻击
- **未选择的塔**: 变成棕色障碍物,阻挡敌人行走

#### 第三步: 开始波次
- 点击顶部绿色"**开始第X波**"按钮
- 敌人开始从起点生成
- 此时无法放置新塔,需要等待波次完成

#### 第四步: 波次完成后继续
- 当所有敌人死亡或到达终点后
- 自动解锁放置阶段(木材重置为5)
- 可以再次放置5个塔

### 资源管理

| 资源 | 用途 | 获取方式 |
|------|------|----------|
| 🪵 **木材** | 放置塔(每次1点) | 每波开始时获得5点 |
| 💰 **金币** | 购买道具(未来扩展) | 击杀敌人获得 |
| ❤️ **矿坑生命** | 矿坑被摧毁则游戏结束 | 初始15点,敌人到达终点扣1点 |

## 💎 塔系统

### 4种基础宝石塔

#### 1. 紫水晶(Amethyst) - 高伤害物理攻击
- **颜色**: 紫色
- **特性**: 单体高伤害,攻速较慢
- **适合**: 对付高血量敌人
- **等级属性**:
  - 碎裂: 伤害25, 范围200, 攻速1200ms
  - 有瑕: 伤害30, 范围210, 攻速1200ms
  - 普通: 伤害40, 范围220, 攻速1200ms
  - 无瑕: 伤害55, 范围230, 攻速1200ms

#### 2. 钻石(Diamond) - 高攻速+多目标
- **颜色**: 白色
- **特性**: 同时攻击多个敌人,攻速极快
- **适合**: 清理大量低血量敌人
- **等级属性**:
  - 碎裂: 伤害8, 范围180, 攻速400ms, 多目标3个
  - 有瑕: 伤害10, 范围190, 攻速400ms, 多目标3个
  - 普通: 伤害13, 范围200, 攻速400ms, 多目标4个
  - 无瑕: 伤害17, 范围210, 攻速350ms, 多目标5个

#### 3. 黄玉(Topaz) - 溅射范围伤害
- **颜色**: 黄色
- **特性**: 攻击时对周围敌人造成溅射伤害
- **适合**: 对付密集群聚的敌人
- **等级属性**:
  - 碎裂: 伤害15, 范围160, 攻速1000ms, 溅射半径60
  - 有瑕: 伤害18, 范围170, 攻速1000ms, 溅射半径70
  - 普通: 伤害24, 范围180, 攻速1000ms, 溅射半径80
  - 无瑕: 伤害32, 范围190, 攻速900ms, 溅射半径100

#### 4. 蛋白石(Opal) - 减速效果
- **颜色**: 浅绿色
- **特性**: 攻击时减速敌人,魔法伤害
- **适合**: 配合其他塔使用,延长敌人暴露在火力下的时间
- **等级属性**:
  - 碎裂: 伤害10, 范围200, 攻速800ms, 减速30%
  - 有瑕: 伤害12, 范围210, 攻速800ms, 减速40%
  - 普通: 伤害16, 范围220, 攻速750ms, 减速50%
  - 无瑕: 伤害22, 范围230, 攻速700ms, 减速60%

### 等级系统

| 等级 | 图标 | 说明 |
|------|------|------|
| Chipped(碎裂) | C | 最低等级,基础属性 |
| Flawed(有瑕) | F | 第二等级,属性提升 |

## 🔧 合成系统

### 基础合成(升级)

**规则**: 2个相同类型 + 相同等级的塔 → 升级为更高等级

**操作流程**:
1. 点击顶部橙色"**🔧 合成**"按钮
2. 在合成列表中查看可合成的塔对
3. 点击一对塔选中(会高亮显示)
4. 点击底部"**✓ 确认合成**"按钮
5. 第一个塔升级为高级塔,第二个塔被消耗

**示例**:
- 2个碎裂紫水晶 → 1个有瑕紫水晶
- 2个有瑕钻石 → 1个普通钻石
- 2个普通黄玉 → 1个无瑕黄玉

### 特殊塔合成

**规则**: 特定类型的塔组合 → 生成新的特殊塔

#### 1. 银塔(Silver) - 钻石 + 黄玉
- **特性**: 多目标攻击 + 溅射伤害
- **属性**: 伤害20, 范围200, 攻速600ms, 多目标3个, 溅射半径80
- **适合**: 同时清理大量敌人

#### 2. 孔雀石(Malachite) - 黄玉 + 蛋白石
- **特性**: 溅射伤害 + 减速效果
- **属性**: 伤害18, 范围190, 攻速900ms, 溅射半径90, 减速50%
- **适合**: 控制+范围伤害

#### 3. 星红宝石(Star Ruby) - 紫水晶 + 钻石
- **特性**: 纯粹伤害(无视护甲) + 多目标攻击
- **属性**: 伤害45, 范围220, 攻速800ms, 多目标4个
- **适合**: 对付高护甲坦克敌人

**操作流程**:
1. 点击顶部"**🔧 合成**"按钮
2. 在"特殊塔配方"区域查看材料是否集齐
3. 如果材料已集齐,显示绿色"**✅ 合成XX塔**"按钮
4. 点击按钮直接合成
5. 材料塔被消耗,新特殊塔出现在存储区

## 👾 敌人系统

### 敌人类型

#### 1. 基础敌人(Basic)
- **颜色**: 蓝色
- **生命**: 50
- **速度**: 60
- **护甲**: 0
- **魔抗**: 0
- **奖励**: 10金币
- **特点**: 最普通的敌人,容易被击杀

#### 2. 快速敌人(Fast)
- **颜色**: 绿色
- **生命**: 35
- **速度**: 100
- **护甲**: 0
- **魔抗**: 0
- **奖励**: 15金币
- **特点**: 移动速度快,但生命值低

#### 3. 坦克敌人(Tank)
- **颜色**: 红色
- **生命**: 150
- **速度**: 40
- **护甲**: 5
- **魔抗**: 0.2
- **奖励**: 30金币
- **特点**: 高生命高护甲,需要高伤害或纯粹伤害才能有效击杀

### 敌人路径

**重要**: 敌人必须按照以下路径移动:

```
起点(顶部绿色) → 矿坑(中心黄色) → 终点(右下角红色)
```

- **起点**: 地图顶部绿色格子,敌人生成位置
- **矿坑**: 地图中心黄色格子,**敌人必须经过这里**
- **终点**: 地图右下角红色格子,敌人到达后矿坑生命-1

**策略提示**: 
- 在矿坑周围建造密集的塔阵
- 使用障碍物引导敌人走更长的路径
- 确保路径始终连通,否则无法放置塔

## 🌊 波次设计

游戏共有12波,难度逐渐增加:

| 波次 | 敌人配置 | 说明 |
|------|----------|------|
| 1 | 5个基础敌人,间隔2秒 | 教学波,1个紫水晶塔即可清掉 |
| 2 | 8个基础敌人,间隔1.8秒 | 稍微增加数量 |
| 3 | 6个基础+3个快速 | 引入快速敌人 |
| 4 | 8个基础+5个快速 | 敌人数量增加 |
| 5 | 8个基础+6个快速+2个坦克 | 引入坦克敌人 |
| 6-11 | 混合配置,数量递增 | 难度持续提升 |
| 12 | Boss波 | 最终挑战 |

**数值平衡**:
- 第1波设计为可轻松通关,帮助新手熟悉操作
- 后续波次敌人数量和强度逐渐增加
- 需要通过合成高级塔来应对后期挑战

## 💡 策略建议

### 新手入门

1. **第一波**: 
   - 放置5个塔,优先保留紫水晶(高伤害)
   - 将塔建在矿坑附近
   - 确保能轻松清掉5个基础敌人

2. **前期(波次1-4)**:
   - 积累相同类型的塔,准备合成
   - 在矿坑周围形成初步防线
   - 注意保存木材,不要浪费

3. **中期(波次5-8)**:
   - 合成2-3个高级塔(普通或无瑕等级)
   - 考虑合成特殊塔(如银塔处理群怪)
   - 优化迷宫布局,延长敌人路径

4. **后期(波次9-12)**:
   - 至少要有1-2个无瑕等级塔
   - 合成星红宝石对付坦克敌人
   - 合理搭配减速塔和高伤害塔

### 进阶技巧

1. **塔的搭配**:
   - 蛋白石(减速) + 紫水晶(高伤) = 经典组合
   - 钻石(多目标) + 黄玉(溅射) = 清理群怪
   - 星红宝石(纯粹伤害) = 克制高护甲坦克

2. **迷宫建造**:
   - 使用障碍物创造S形或螺旋形路径
   - 让敌人在矿坑附近绕圈
   - 确保路径不要太长导致敌人堆积

3. **合成时机**:
   - 不要急于合成,先积累材料
   - 优先合成关键位置的塔
   - 特殊塔在需要时再合成(如坦克出现时合成星红宝石)

4. **资源管理**:
   - 每波只放置必要的塔
   - 保留木材用于紧急调整
   - 金币留着未来可能的升级功能

## 🛠️ 技术实现

### 技术栈
- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **渲染引擎**: HTML5 Canvas 2D
- **状态管理**: React Hooks (useState, useRef)
- **寻路算法**: BFS (广度优先搜索)

### 核心机制

1. **BFS寻路**: 
   - 实时计算敌人路径
   - 支持动态障碍物(放置塔后重新计算)
   - 确保路径经过矿坑

2. **游戏循环**:
   - 使用requestAnimationFrame实现60fps
   - 分离逻辑更新和渲染
   - 使用useRef避免不必要的重渲染

3. **碰撞检测**:
   - 圆形碰撞(塔攻击范围)
   - 距离平方比较优化性能
   - 子弹追踪敌人

4. **伤害计算**:
   - 物理伤害: `damage * (1 - armor/(armor+10))`
   - 魔法伤害: `damage * (1 - magicResist)`
   - 纯粹伤害: 无视减免

## ❓ 常见问题

**Q: 为什么点击已有塔会弹出"木材已用完"?**  
A: 这是旧版本的bug,已修复。现在点击已有塔会打开合成对话框或选择保留。

**Q: 敌人为什么不经过矿坑?**  
A: 已修复路径算法,确保敌人必须经过中心矿坑再到终点。

**Q: 为什么显示"材料已集齐"但不能合成?**  
A: 需要2个相同类型+等级的塔才能进行基础合成。特殊塔合成为每个配方添加了"合成"按钮。

**Q: 如何查看更多塔的详细信息?**  
A: 打开合成对话框,可以看到所有塔的属性说明。
