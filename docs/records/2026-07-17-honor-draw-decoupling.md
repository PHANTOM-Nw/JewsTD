# 中發白定期抽取与手牌解耦

日期：2026-07-17

## 规则

- 每个建造轮在三选一后都进入 `resolving_hand`，公开剩余数牌花色并保留 1 张下回合手牌；点数继续保密，其余数牌回池。
- 第 2、4、6、8、10、12 建造轮在留牌完成后自动进行一次中發白抽取，成功率固定为 50%。
- 抽取不读取数牌花色组成，不要求旧手牌仍在剩余牌中，不消耗任何数牌，也不会清空或替换刚保留的手牌。
- 成功后中、發、白等概率（各 1/3）进入功能牌区；失败不增加功能牌。界面在 `ready` 阶段显示本次结果。

## 实现

- `src/game/config/mahjong.ts` 保存每 2 轮的间隔和固定 50% 成功率，并提供可注入随机源的功能牌抽取函数。
- `src/game/engine/mahjongRoundFlow.ts` 以纯事务同时返回留牌、回池、功能牌与抽取结果；失败输入不修改原集合。
- `useGameEngine` 只维护从 1 开始的麻将建造轮编号并提交纯事务结果，不再提供旧赌博动作。
- UI 删除“留牌或赌博”的二选一按钮与花色组成概率，只在偶数轮提示留牌后会自动抽取，并显示成功或失败结果。

## 不变量与验证

- 奇数轮不会读取功能牌随机源；偶数轮先以固定 50% 判定，成功后才读取第二次随机选择中、發、白。
- 无论抽取成功或失败，所选手牌实体仍由 `heldTile` 持有，未选实体回池，功能牌继续位于 108 张数牌集合之外。
- 测试覆盖第 2、4、6、8、10、12 轮调度、50% 边界、中發白三段等概率映射、失败随机调用次数、旧手牌是否参与建造的两种剩余牌组成，以及 108 张实体守恒。
