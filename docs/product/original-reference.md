# 经典宝石 TD 核心玩法调研材料

> 本文是项目早期收集的设计参考，未附完整来源校验，不代表当前实现或产品承诺。当前玩法以 `gameplay.md` 和源码为准。

一、游戏基本信息
1.1 起源与发展历史
宝石TD(Gem Tower Defense)是一款经典的塔防类RPG自定义地图,最初诞生于《魔兽争霸3:冰封王座》平台。根据资料显示:
起源时间: 约2005年左右开始流行
原作者: 澳大利亚开发者bryvx(美西)
汉化版本: 由vinxu等译者进行中文本地化
最终版本: War3版本最终停留在v3.1(2010年8月)
Dota2移植: 后移植至Dota2游廊平台,由四七工作室开发维护
最新版本: Dota2版本截至2023年7月更新至Ver12.7
1.2 游戏基本目标与胜利/失败条件
胜利条件:
成功抵御所有波次的怪物进攻(通常为42波或50波,取决于模式)
保护矿坑(Mine)的生命值不被耗尽
失败条件:
矿坑生命值归零
怪物突破防线到达终点
游戏模式:
正常模式: 42波怪,3个Boss(第14、28、42波后出现),玩家同步进行
竞赛模式: 50波怪,2个Boss(第25、50波),无需等待其他玩家清完怪即可开始下一轮
难度选择: 简单、普通、困难、极难四个等级
1.3 Dota2游廊版本特点
相比War3原版,Dota2版本有以下改进:
多人合作机制: 从各自为战改为团队协作模式
地图扩展: 更大的地图空间,更多的发挥余地
动态难度: 难度随游戏人数自动调整
支持人数: 1-4人合作(War3版支持1-8人)
新增内容: 更多原创宝石、防御塔和技能系统
画质提升: 使用Source 2引擎,视觉效果大幅提升
Bug修复: 解决了War3编辑器下的诸多技术限制
二、塔的系统(核心特色)
2.1 八种基础宝石塔
宝石TD的核心是8种基础宝石塔,每种具有独特的攻击特性和定位:
宝石名称	英文代号	攻击特性	特殊效果	适用场景
紫水晶	A (Amethyst)	对空专用	射程最远	防空主力
钻石	D (Diamond)	对地专用	暴击几率	单体高伤
黄玉	T/Y (Topaz)	多重攻击	同时攻击多个目标	群体输出
蛋白石	O (Opal)	辅助型	攻速光环(范围内塔加速)	团队增益
红宝石	R (Ruby)	范围攻击	溅射伤害	清群怪
蓝宝石	S (Sapphire)	单体攻击	冰霜缓速,攻速最慢	控制减速
翡翠	E (Emerald)	单体攻击	毒素缓速+持续伤害	持续削弱
海蓝宝石	Q (Aquamarine)	单体攻击	攻速极快,射程最短	快速输出
攻击类型说明:
物理伤害: 受敌人护甲减免
魔法伤害: 受敌人魔抗影响
纯粹伤害: 无视护甲和魔抗(部分特殊塔具备)
2.2 塔的等级系统
宝石塔共有7个品质等级,从低到高依次为:
Chipped(碎裂) - 初始默认等级
Flawed(破碎/瑕疵)
Normal(普通)
Flawless(无暇)
Perfect(完美)
Giant(巨型) - 后期高级形态
BRYVX之石 - 最高级隐藏形态
升级机制:
基础合成: 2个同等级同种类宝石可合并提升1级
超级合成: 4个同等级同种类宝石可提升2级
概率提升: 通过矿坑升级可提高高级宝石出现概率
矿坑升级后的出塔概率(满级时):
Chipped: 0%
Flawed: 30%
Normal: 30%
Flawless: 30%
Perfect: 10%
物理系宝石塔成长: 每提升一级,面板攻击力增加10%
2.3 特殊合成塔系统
这是宝石TD最具特色的机制之一。通过组合不同种类的宝石,可以创造出功能强大的特殊塔。
常见特殊塔配方:
1. 银(Silver)
合成公式: 碎裂蓝宝石(B1) + 碎裂黄玉(T1) + 碎裂钻石(D1)
可升级: 纯银 → 银骑士
特性: 喷溅伤害+缓速效果
评价: 非常实用的综合型塔,前期核心
2. 孔雀石(Malachite)
合成公式: 碎裂翡翠(E1) + 碎裂海蓝宝石(Q1) + 碎裂蛋白石(O1)
可升级: 鲜艳孔雀石 → 巨大孔雀石
特性: 多重攻击(初始3目标,升级后5目标)
基础属性: 12攻击力(初始) / 36攻击力(巨大)
评价: 前期群体物理输出主力
3. 星彩红宝石(Star Ruby)
合成公式: 碎裂紫晶(A1) + 碎裂红宝石(R1) + 瑕疵红宝石(R2)
可升级: 血之星 → 火之星
特性: 前两阶为范围法术燃烧,第三阶变为多重物理攻击
评价: 法伤转物伤的混合型塔
4. 玉(Jade)
合成公式: 瑕疵蓝宝石(S2) + 蛋白石(O) + 翡翠(E)
可升级: 中国玉 → 吉祥的中国玉
特性: 几率击晕、额外金钱奖励、暴击能力
评价: 经济型控制塔
5. 红水晶(Red Crystal)
合成公式: 瑕疵紫晶(A2) + 红宝石(R) + 无暇翡翠(E4)
可升级: 红色刻面石 → 芙蓉石
特性: 仅对空,极高伤害,范围扣防灵气
特殊: 第一等级可花费200金币换位置一次,升级后不可换位
评价: 顶级防空塔
6. 黑暗翡翠(Dark Emerald)
合成公式: 完美翡翠(E5) + 无暇蓝宝石(S4) + 瑕疵黄玉(T2)
可升级: 迷人翡翠
特性: 第一等级带击晕,第二等级增加暴击
评价: 单体控制+爆发,打Boss关键塔
7. 鸡血石(Bloodstone)
合成公式: 紫晶(A) + 无暇海蓝宝石(Q4) + 完美红宝石(R5)
可升级: 古代鸡血石
特性: 第一等级范围燃烧伤害,第二等级单体普攻+火焰冲击
评价: 持续法伤输出
8. 铀238(Uranium-238)
合成公式: 蓝宝石(S) + 瑕疵蛋白石(O2) + 完美黄玉(T5)
可升级: 铀235
特性: 升级前范围伤害+缓速,升级后多重攻击
评价: 中后期强力输出
9. 金(Gold)
合成公式: 瑕疵钻石(D2) + 无暇紫晶(A4) + 完美紫晶(A5)
可升级: 埃及金
特性: 暴击能力+单体扣防效果
评价: 破甲辅助输出
10. 粉红钻石(Pink Diamond)
合成公式: 黄玉(T) + 钻石(D) + 完美钻石(D5)
可升级: 巨型粉红钻石
特性: 暴击能力,升级后效果提升
评价: 高爆发单体输出
11. 黑色猫眼石(Black Cat's Eye)
合成公式: 海蓝宝石(Q) + 无暇钻石(D4) + 完美蛋白石(O5)
可升级: 珊瑚石
特性: 大范围攻击灵气,可花费200金币换位置
评价: 团队增益型塔
12. 帕拉依巴碧玺(Paraiba Tourmaline)
合成公式: 瑕疵翡翠(E2) + 瑕疵海蓝宝石(Q2) + 无暇蛋白石(O4) + 完美海蓝宝石(Q5)
可升级: 精心雕琢的帕拉依巴碧玺
特性: 地面部队扣防灵气+几率寒冰爆
评价: 高级辅助控制塔
2.4 隐藏塔系统(彩蛋)

2. 玛瑙石(Agate) - "430"
合成公式: G5(完美翡翠) + E5(完美海蓝宝石) + Q5(完美蛋白石)

2.5 石板系统(Slab)
石板是一种不阻挡怪物路径的特殊建筑,可以与宝石或其他石板叠加放置。石板合成规则: 只能从单波盖出的五颗宝石中合成,不能跨波次累积常见石板配方:1. 浮空石板(Floating Slab)
配方: A3(普通紫晶) + E2(瑕疵翡翠) 或 O2(瑕疵蛋白石)
特性: 兼具对空对地能力
2. 滞留石板(Stun Slab)
配方: T3(普通黄玉) + A2(瑕疵紫晶) 或 S2(瑕疵蓝宝石)
特性: 将怪物击晕2秒,冷却2.5秒
3. 远古石板(Ancient Slab)
配方: 浮空石板 + 滞留石板
特性: 复合功能石板
4. 缓速石板(Slow Slab)
配方: S3(普通蓝宝石) + D2(瑕疵钻石) 或 E2(瑕疵翡翠) 或 Q2(瑕疵海蓝宝石)
特性: 强力减速效果
5. 蛋白石脉板(Opal Vein Slab)
配方: O3(普通蛋白石) + T2(瑕疵黄玉) 或 R2(瑕疵红宝石)
特性: 攻速增益
6. 幽灵石板(Ghost Slab)
配方: 缓速石板 + 蛋白石脉板
特性: 减速+攻速双重效果
石板进阶:
毒藤石板、法术石板等可进一步合成为遗迹石板或远古石板
石板可与其他塔叠加,形成复合型防御体系
2.6 天命出塔机制
天命出塔(Destiny Spawn)是Dota2版本引入的重要机制:
当玩家一个回合内放置的5个防御塔满足特定合成公式时
系统会提示可以升级为高级防御塔
可直接合成特殊塔,无需逐波保留材料
大幅降低运气成分,提高策略性
举例: 若一回合内同时出现蓝宝石、黄玉、钻石,可直接合成银塔
三、敌人系统
3.1 敌人种类与特点
基础敌人分类:
地面单位: 常规近战/远程怪物
空中单位: 飞行怪物,不受迷宫地形影响
Boss单位: 高血量高攻击的特殊敌人
敌人特殊属性:
护甲(Armor): 减免物理伤害
魔抗(Magic Resistance): 减免魔法伤害
移动速度: 影响通过迷宫的时间
特殊抗性: 部分敌人免疫特定效果(如魔免、眩晕免疫)
3.2 波次设计规律
标准模式(42波):
总波数: 42波
空军波: 每4的倍数波次(4、8、12、16...40)出现空中单位
Boss波: 第14、28、42波后出现Boss
竞赛模式(50波):
总波数: 50波
Boss波: 第25、50波出现Boss(第25波实际为测试者)
难度递增曲线:
前期(1-10波): 敌人血量低,数量少,熟悉机制阶段
中期(11-30波): 敌人属性提升,出现混合兵种,Boss考验阵容
后期(31-42/50波): 高强度敌人,Boss血量可达数百万
3.3 Boss机制
Boss特点:
超高血量: 后期Boss血量可达300万甚至更高
高额伤害: 对矿坑造成"血量%×80+10"的伤害
特殊技能: 可能具备回血、重跑、减甲等机制
Boss应对策略:
优先击杀高攻击力Boss,减少矿坑损失
使用黑暗翡翠等眩晕塔控制Boss
搭配多种伤害类型(物理+魔法)突破抗性
重跑者机制(某些版本):
Boss每重跑一次减少10点护甲
重跑4次后停止回血5000点
3.4 敌人速度与计算
速度计算公式:
最低速度限制为100
黄菜(黄玉系列)减速最后计算
石像鬼减速30%
B1-B5齐全可减速600
速度叠加示例:
将怪物减速到400后,再进入黄玉范围即达到最低速度100
四、资源/宝石系统(核心特色)
4.1 资源类型
宝石TD采用双资源系统:1. 木材(Wood/Lumber)
用途: 建造防御塔的唯一资源
获取: 每波开始时自动获得5个木材
消耗: 每个塔消耗1个木材
特点: 固定收入,无法储存,每波必须用完
2. 金币(Gold/Money)
用途:
购买生命(10金币+1命)
升级矿坑(提高高级宝石出现概率)
购买完美宝石出现概率
购买石板出现概率
特殊塔换位(如红水晶)
获取:
击杀怪物获得
玉等特殊塔提供额外金钱
评分奖励
特点: 可积累,用于长期投资
4.2 矿坑系统(Mine)
矿坑是地图中央的核心建筑,具有以下功能:生命值机制:
初始生命: 15点
最大生命: 可通过购买提升至50点
损失规则:
前期漏怪: 每只-1生命
后期漏怪: 每只扣除当前关卡一半的生命
失败条件: 生命值归零则游戏结束
矿坑升级选项(4个按钮):
购买生命: 10金币+1生命值
增加宝石品质: 提升建造时出现高级宝石的概率
购买完美宝石概率: 极低概率直接出现完美宝石
购买石板概率: 极低概率直接出现石板,无需合成
升级成本: 越高级升级花费越多金币
4.3 经济系统设计
收入来源:
基础收入: 每波固定5木材
击杀奖励: 消灭怪物获得金币
特殊奖励: 玉塔的额外金钱效果
评分奖励: 高评分结算获得更多资源
支出项目:
矿坑升级: 提高出塔品质的长期投资
生命购买: 应急保命手段
概率购买: 赌运气的高级投资
经济管理策略:
前期优先保证防御,适度升级矿坑
中期平衡发展与防守
后期根据局势决定是否买命
4.4 宝石与塔的关系
核心循环:
每波获得5木材 → 建造5个基础塔 → 选择保留1个或合成 → 
其余4个变成迷宫岩石 → 下波重复
宝石作用:
作为建造材料形成防御塔
未保留的塔转化为迷宫地形
通过合成创造更强大的特殊塔
高等级宝石提供更强的基础属性
五、特殊机制
5.1 技能系统
Dota2版本新增:
英雄技能: 8种英雄技能可选
强力技能: 多达50种技能供选择
技能类型: 主动释放型和被动增益型
技能获取方式:
购买英雄只是购买技能槽位
技能需要单独购买
颜色对应技能槽数量(蓝色和白色英雄可忽略)
常见技能类型:
主动技能: 风暴之锤、回到过去等
被动技能: 闪避、守护、回春等
定位技能: VS乾坤小挪移等
祈祷许愿类: 看脸的技能,谨慎购买
技能叠加规则:
毒刺: 可任意叠加(翡翠系列G1-G6和玉系列)
灼烧: 可任意叠加(星彩、火山、古代鸡血石等)
腐蚀: 无视魔免,不同级可叠加
虚弱: 不同级可叠加(大小碧玺)
折翼: 不同级可叠加,折翼2无视魔免
光环: 不同级可叠加,EH光环直接达到极限攻速500
望远: 可叠加
注意: 同级非Max效果不可叠加,只有Max级别可叠加
5.2 光环效果
蛋白石光环:
一定范围内的塔获得攻速加成
不同等级光环可叠加
EH光环可使攻速达到极限500
其他光环塔:
黑色猫眼石: 大范围攻击灵气
帕拉依巴碧玺: 地面部队扣防灵气
红水晶: 对空范围扣防灵气
光环范围:
九宫格对角线位置的塔可能吃不到光环
Max光环范围为鼠标移上去显示的范围
略小于海洋青玉(望远镜)的加射程范围
5.3 元素克制系统
伤害类型相克:
物理伤害 vs 护甲: 护甲越高,减免越多
魔法伤害 vs 魔抗: 魔抗越高,减免越多
纯粹伤害: 无视护甲和魔抗(部分特殊塔具备)
塔与怪物装甲关系:
不同塔的攻击形态对不同装甲有加成或克制
需要根据敌人类型选择合适的塔组合
典型搭配:
高护甲敌人: 使用魔法伤害塔(红宝石、星彩等)
高魔抗敌人: 使用物理伤害塔(钻石、孔雀石等)
混合抗性: 物理+魔法双修阵容
5.4 随机性元素(RNG)
随机性来源:
出塔随机:
每波5个塔的种类完全随机
初始100%为碎裂等级
升级矿坑可提高高级塔概率,但仍不确定
合成随机:
隐藏塔需要同一回合集齐,概率极低
天命出塔虽有提示,但依赖当波随机结果
技能触发:
傲娇虾妹石30%概率不攻击
连锁闪电20%概率触发
叉状闪电50%概率触发
暴击、眩晕等均为概率触发
购买概率:
完美宝石出现概率极低
石板直接出现概率极低
RNG应对策略:
多波次规划,为合成预留材料
建立备选方案,不过度依赖单一塔
利用降级功能调整宝石等级
合理投资矿坑,提高整体质量
六、地图布局
6.1 地图规格
地图尺寸: 37×37格的开放地图布局特点:
无固定路径: 没有预设的怪物行进路线
玩家自建迷宫: 通过放置未保留的塔形成岩石障碍
7个固定点: 怪物必须经过包括起点和终点在内的7个固定位置
自由度高: 玩家可创造性地设计迷宫形状
6.2 建造区域限制
建造规则:
每波只能建造5个塔
必须立即决定保留哪个塔
未保留的4个塔自动转化为迷宫岩石
岩石一旦放置不可移除(除非使用特殊换位技能)
中心区域:
地图中央有三栋建筑物:
矿坑(Mine): 核心防御目标
特殊宝石合成配方商店: 显示特殊塔合成配方
石板合成配方商店: 显示石板合成配方
可用空间计算:
中间区域约5×5=25格
扣除3栋建筑占用约6格
实际可用约19格用于建塔
6.3 迷宫设计原则
有效迷宫特征:
延长路径: 使怪物行走距离最大化
多层环绕: 让怪物多次经过火力覆盖区
三进三出: 经典的高效迷宫设计
斜道优化: PC版怪物走斜线,手游版Z字型,斜阵比方阵优秀
迷宫工具:
社区开发了专门的迷宫设计器
可保存和分享.TD格式的迷宫文件
支持切换玩家位置查看不同视角
6.4 格子系统
网格特性:
标准正方形格子
每个塔占据1个格子
石板可与塔叠加在同一格子
XY轴两格或XY轴一格加斜一格为常见光环覆盖范围
七、游戏流程
7.1 典型游戏时长
单局时长:
正常模式: 约40-60分钟(42波)
竞赛模式: 约50-70分钟(50波)
影响因素: 玩家熟练度、难度选择、是否漏怪重来
7.2 波次数量与节奏
波次分布:
前期(1-10波): 每波间隔短,快速建立基础防御
中期(11-30波): 间隔适中,需要思考和调整
后期(31-42/50波): 每波压力大,可能需要暂停思考
每波流程:
获得5个木材
立即建造5个塔
短暂思考时间决定保留哪个
如有相同塔可合成升级
如有特殊组合可合成特殊塔
其余4个塔变为岩石
怪物开始进攻
清完怪后进入下一波
7.3 难度递增曲线
敌人强度增长:
血量: 指数级增长,后期Boss达数百万
攻击力: 线性增长,后期对矿坑威胁极大
速度: 逐渐加快,考验减速塔配置
数量: 逐渐增多,考验群体输出能力
玩家压力变化:
前期: 轻松,主要学习机制
中期: 开始紧张,需要合理规划
后期: 高压,容错率极低
难度选择影响:
简单: 适合新手熟悉游戏
普通: 标准体验
困难: 挑战性强
极难: 需要完美操作和运气
7.4 阶段性目标
前期(1-10波):
建立基础防御体系
至少合成1-2个银或孔雀石
初步构建迷宫框架
适度升级矿坑
中期(11-30波):
完善特殊塔组合
应对第一个Boss(14波)
平衡物理和魔法输出
准备防空力量
后期(31-42/50波):
全力输出,清理高强度敌人
应对最终Boss
必要时购买生命保命
争取高评分通关

九、宝石TD的核心特色总结
9.1 区别于其他塔防游戏的独特之处
1. 无固定路径的迷宫建造
大多数塔防游戏有预设路线
宝石TD让玩家自己创造迷宫
未保留的塔自动变成障碍物
高度自由的策略空间
2. 随机性与策略性的完美结合
每波5个塔完全随机,考验应变能力
特殊塔合成需要跨波次规划
天命出塔机制平衡运气成分
既有趣味性又有深度
3. 丰富的塔系统和合成树
8种基础塔各有特色
12+种特殊塔提供多样化选择
7个隐藏塔作为彩蛋和高追求
石板系统增加战术层次
4. 双资源系统的精妙设计
木材强制每波用完,避免囤积
金币用于长期投资和应急
矿坑升级提供成长性
经济决策影响游戏进程
5. 物理与魔法双输出体系
需要考虑敌人护甲和魔抗
不同类型塔互补配合
纯粹伤害塔作为终极手段
伤害类型克制增加策略深度
6. 光环与技能叠加机制
不同等级光环可叠加
多种debuff可同时生效
蛋白石攻速光环改变战局
技能系统(Dota2版)增加变数
7. 高风险高回报的隐藏塔
同一回合集齐的苛刻条件
超强DPS但概率触发不稳定
作为彩蛋增加探索乐趣
高手向的挑战目标
9.2 游戏设计亮点
节奏控制:
每波之间有时间思考
不会手忙脚乱
适合策略性游戏
学习曲线:
新手可从简单难度入门
逐步解锁复杂机制
高手有无尽追求(隐藏塔、高评分)
重玩价值:
随机出塔保证每局不同
多种合成路线可尝试
不同难度和挑战模式
多人合作增加社交性
平衡性:
没有绝对最强的塔
需要根据局势灵活搭配
物理魔法相互制衡
RNG与技巧并重
9.3 对塔防游戏设计的启示
宝石TD的成功要素:
自由度与限制的平衡: 无固定路径但每波只能建5个塔
随机性与可控性的结合: 随机出塔但可跨波次规划合成
简单规则产生复杂策略: 基础机制简单但组合变化无穷
短期决策与长期规划的统一: 每波即时决策+多波合成计划
个人技巧与团队协作的融合: Dota2版的合作模式
核心玩法的深度挖掘: 围绕"宝石合成"这一核心做深做透

结语
宝石TD作为一款经典的塔防游戏,其成功在于将简单的核心机制(随机出塔、宝石合成、迷宫建造)做到了极致。它既保留了塔防游戏的基本乐趣,又通过独特的设计创造了深度的策略空间。无论是War3时代的经典版本,还是Dota2平台的现代化改进,宝石TD都证明了优秀的游戏设计可以跨越平台和时代,持续吸引玩家。对于游戏设计师而言,宝石TD提供了宝贵的经验:在限制中创造自由,在随机中保持策略,在简单中蕴含深度。这些设计理念值得在任何类型的游戏开发中借鉴和应用。
